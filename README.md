# Swiggy-Restaurant-Data 

This project focuses on analyzing Swiggy restaurant data to uncover meaningful business insights Exploratory Data Analysis (EDA) was performed to identify cuisine trends, pricing patterns, and customer preferences Interactive Power BI dashboards were developed to visualize restaurant performance and market distribution and Key metrics such as ratings, cost distribution, and popular cuisines were analyzed to support data driven decisions The project helps stakeholders understand customer behavior and optimize restaurant strategies for better growth
